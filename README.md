# Panther

Experimental HTTP/2 server for Rack.

**This is NOT production ready, just a code spike.**

## Usage

```shell
% git clone https://github.com/jodosha/panther.git && cd panther
% bundle && cd example 
% ./server
```

```shell
% curl -I -k --http2 https://localhost:7152
```
